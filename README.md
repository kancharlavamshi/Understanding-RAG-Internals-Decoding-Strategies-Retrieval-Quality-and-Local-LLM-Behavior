This project demonstrates how to build a Retrieval-Augmented Generation (RAG) system completely from scratch using:

🤖 Local Large Language Models (TinyLlama / Transformers)

🔍 FAISS Vector Similarity Search

🧠 BGE Embeddings (GPU Accelerated)

📄 PDF Course Material

⚡ Fully Local Inference (No APIs)

The goal was not just to build a RAG system — but to understand how it works internally, including retrieval quality, decoding behavior, and grounding strategies. 

🎯 Project Objectives

Implement an end-to-end RAG pipeline

Understand embedding-based retrieval

Analyze decoding strategies (temperature, do_sample, etc.)

Prevent hallucinations using strict grounding

Study how generation parameters affect output stability

Build everything locally on GPU
